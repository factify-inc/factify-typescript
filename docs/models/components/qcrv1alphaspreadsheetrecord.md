# QcrV1alphaSpreadsheetRecord

SpreadsheetRecord is the QCR representation of an ingested spreadsheet
 workbook (XLSX, CSV, Parquet). Immutable once extracted. Source-format-agnostic.

 Hierarchy: SpreadsheetRecord 1:M Sheet 1:M DataTable/SheetImage/SheetChart.
 The semantic core is DataTable: named, typed columns applied to ordered rows.
 The extractor identifies headers and schemas so consumers don't have to.

## Example Usage

```typescript
import { QcrV1alphaSpreadsheetRecord } from "@factify/sdk/models/components";

let value: QcrV1alphaSpreadsheetRecord = {};
```
