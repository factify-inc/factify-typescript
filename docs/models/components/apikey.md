# ApiKey

ApiKey represents an authentication credential for programmatic API access.

## Example Usage

```typescript
import { ApiKey } from "@factify/sdk/models/components";

let value: ApiKey = {
  createdAt: new Date("2025-10-04T19:57:22.198Z"),
  id: "key_01jd4h5mck9gq6zrp8bn2t4w3x",
  isActive: true,
  keyPrefix: "ffy_prod_01jd4h5...",
  name: "Production",
  organizationId: "org_01h2xcejqtf2nbrexx3vqjhp41",
};
```
